# 🍽️ Prince Palace - Restaurant Website

Prince Palace is a visually appealing and responsive restaurant website built using HTML and CSS. It is designed to provide users with an engaging experience to explore restaurant services, view offerings, and place food orders.

---

## 🚀 Features

- 🏠 Attractive Home Page with hero section
- 🍕 Services Section (Restaurant, Bulk Ordering, Food Ordering)
- 👥 Clients Section showcasing popular brands
- 📞 Contact Form for user interaction
- 🎨 Clean and modern UI design
- 📱 Responsive layout (basic)

---

- HTML5
- CSS3
- (Optional: JavaScript if added later)
